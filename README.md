# Tienda Gamer

Proyecto de Comercio Electrónico para Componentes y PCs Gaming

## Tecnologías Utilizadas

- **Frontend:** HTML, CSS, JavaScript (en desarrollo)
- **Backend:** Node.js + Express.js
- **Base de Datos:** SQLite
- **Servidor:** Ubuntu Server 25 en Oracle VirtualBox
- **Control de versiones:** Git + GitHub
- **Herramientas:** Git Bash + SSH + WinSCP + Postman

## Requisitos

- Oracle VirtualBox instalado en la PC host
- Máquina Virtual con **Ubuntu Server 25**
- Node.js y npm instalados en la VM
- Git Bash en la PC host
- Conexión SSH desde Git Bash a la VM
